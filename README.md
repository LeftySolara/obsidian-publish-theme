# Julianne's Obsidian Publish Theme

This is the CSS used for my [Obsidian Publish site](https://wiki.julianneadams.dev).
It is a custom theme inspired by [selfire1 on GitHub](https://github.com/selfire1/obsidian-publish-css).

This theme is not as extensive as most of the big ones that are out there
and does not cover as many elements.
It has been built from the ground up and is iterated on
as I find more things that I want to change.

If you notice any errors or inconsistencies, feel free to
[send me an email](mailto:julianne@julianneadams.dev) to let me know.
